# betterGPUwin
VBS from betterGPUwin team
Requirements:
- Python 3.10
- Linux or WSL2
- ffmpeg
        make ffmpeg
- Config: Vaof file config/common_config sửa lại tham số

To setup
    make install 
or
    pip install -r setup.txt

To make new collection:

        make init
or
        python src/init.py 

To run analyze:

        python src/analyze/analyze.py --id 02082013
